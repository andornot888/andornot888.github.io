Java中的substring和SQL中substr的区别

发表于 2021-11-28 更新于 2022-03-18 分类于 后端 Java MySQL
本文字数： 512 阅读时长 ≈ 1 分钟

java的substring(a,b)
从下标a开始，截取到下标b，左闭右开，也就是包含下标a的值，但不包含下标b的值。

或者substring(a)
去除从下标0开始，到下标a的这一段，返回剩下的那一段

SQL中的substr(str,a,b)
第一个参数：字符串
第二个参数：从哪里开始取 (下标从1开始，不是0)
第三个参数：取几位

从5开始，包括5，返回剩下的一段 (下标从1开始，不是0)
