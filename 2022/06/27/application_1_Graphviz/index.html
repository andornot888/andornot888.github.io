作图工具Graphviz

发表于 2022-06-27 分类于 应用使用教程 Graphviz
本文字数： 2.2k 阅读时长 ≈ 2 分钟

Graphviz由一种被称为DOT语言的图形描述语言 ，与一组可以生成和/或处理DOT文件的工具组成：

  • dot：一个用来将生成的图形转换成多种输出格式的命令行工具。其输出格式包括PostScript，PDF，SVG，PNG，含注解的文本等等。

  • neato：用于sprint model的生成（在Mac OS版本中称为energy minimized）。

  • twopi：用于放射状图形的生成 。

  • circo：用于圆形图形的生成 。

  • fdp：另一个用于生成无向图的工具 。

  • dotty：一个用于可视化与修改图形的图形用户界面程序 。

  • lefty：一个可编程的（使用一种被EZ影响的语言）控件，它可以显示DOT图形，并允许用户用鼠标在图上执行操作。Lefty可以作为MVC模型的使用图形的GUI程序中的视图部分 。
